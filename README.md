## jiao-vs-yelp

## about my project
This is a database that tracks my rating of restaurants and compares them against their public ratings on Google and Yelp. 
I'm using it as a scientific way to prove or disprove my friends' claims that I have bad taste in food.


## database schema + CRUD operations
Each entry has a id, restaurant name, cuisine type, my personal rating, and the public rating. 
- Create: you can add a new entry 
- Read: the current data is displayed in an HTML file that can be searched and sorted
- Update: you can update the values of an existing entry without making a new one
- Delete: you can delete an entry and the id is automatically reindexed

## running the app
Just run the python script. The database and html file will be generated automatically and can be viewed locally.
